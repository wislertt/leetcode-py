# Pascal's Triangle

**Difficulty:** Easy
**Topics:** Array, Dynamic Programming
**Tags:** grind-75

**LeetCode:** [Problem 118](https://leetcode.com/problems/pascals-triangle/description/)


## Problem Description

Given an integer numRows, return the first numRows of Pascal&#39;s triangle.

In Pascal&#39;s triangle, each number is the sum of the two numbers directly above it as shown:

&nbsp;
Example 1:
Input: numRows = 5
Output: [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]
Example 2:
Input: numRows = 1
Output: [[1]]

&nbsp;
Constraints:


	1 &lt;= numRows &lt;= 30
